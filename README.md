# review_analysis_filluminate

## 초기 실행
```
git clone https://github.com/yoohengu/review_analysis_filluminate.git
cd review_analysis_filluminate

uv sync
```

## 작업 전
```
deactivate  # 가상환경 끄기(보통 할 필요 없음)
source .venv/Scripts/activate  # 가상환경 접속 (이것도 보통 할 필요 없을듯)
git pull
```

## 작업 후
```
git pull
git add .
git commit -m '작업 내용'
git push
```
- `push` 하기 전에는 반드시 팀원들에게 얘기하고 `push` 해주세요

## ㅇㅇ
- 작업 전에는 반드시 `git pull` 해주세요
- 데이터 파일은 최상위 디렉토리의 `data` 폴더에 저장 후 작업해주세요
- 작업은 최대한 본인 폴더에서 해주세요
